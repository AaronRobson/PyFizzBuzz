# Getting Started
https://blog.codinghorror.com/why-cant-programmers-program/

## Install dependencies
Install `python3` including `pip3`.

## Run
```bash
make run
```
or:
```bash
python3 fizzbuzz
```

## Run dev checks

### Install Dev dependencies
```bash
pip3 install flake8 pep8-naming
```

### Lint
```bash
make check
```

### Unittest
```bash
make test
```

### Run all
```bash
make
```
